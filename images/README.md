# BAM'S BARBERS — Images Folder

Drop your images here, then upload this whole folder to GitHub.

## How to upload
1. Go to **github.com/roy-habib/bams-barbers**
2. Click **+ → Upload files**
3. Drag the images below into the upload area
4. Set the folder path to `images/` before committing

---
